You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /prime -- returns primes from num1 to num2: request should be: /prime?num1=2&num2=100
  • /programmer -- returns full name of programmers first name passed as parameter, accepted values are:
    • Alexandra
    • Irvin
    • Kyle
    • Ex of request: /programmer?fn=Irvin
